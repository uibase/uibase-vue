一部のアイコンはエイリアス名で指定可能です

```vue
<base-icon name="!" />
<base-icon name="?" />
<base-icon name="*" />
<base-icon name="x" />
```

`size`でサイズを指定できます。

```vue
<base-icon name="!" size="40px" />
```

`color`で色を指定できます

```vue
<base-icon name="!" size="40px" color="primary" />
```
